# MALINA (Nawigacja po "muzeum")
Aplikacja mobilna na Androida realizująca nawigację wewnątrz budynku na przedmiot **M**obilne **A**plikacje **L**okalizacyjne **i** **Na**wigacyjne

Aplikacja działa za pomocą odczytywania kodów QR w kolejnych pomieszczeniach i kierowania do kolejnego pomieszczenia i jego kodu QR.
